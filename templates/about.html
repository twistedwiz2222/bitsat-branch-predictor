ABOUT BITSAT BRANCH PREDICTOR
Branch Cutoffs

The branch cutoffs displayed in this predictor are based on the 2024 BITSAT admission data, with projections for 2025. These cutoffs represent the minimum score required to be eligible for each branch at respective campuses.

Note that these projections for 2025 are estimates based on past trends and may vary due to:

  • Number of applicants
  • Overall performance of test-takers
  • Available seats in each branch

How to Use the Predictor

Simply enter your BITSAT score in the input field on the home page and click the "Find My Eligible Branches" button. The predictor will display all branches across all three BITS campuses (Pilani, Goa, and Hyderabad) for which you would be eligible based on the cutoff scores.

The branches are sorted by cutoff score, with the most competitive branches (highest cutoffs) appearing first.

Important Note: This predictor is meant to be used as a guidance tool based on historical data and projections. The actual cutoffs for 2025 may vary from the estimated values shown here.
